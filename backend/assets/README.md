# Logo Gear Assets

## Required Images for DC Generation

To enable automatic image embedding in DC files, please add the following images to this directory:

### 1. Logo Gear Company Logo
- **Filename**: `logo-gear.png`
- **Description**: The blue oval "Logo Gear" company logo
- **Recommended size**: 200x100 pixels
- **Format**: PNG with transparent background

### 2. Authorized Signatory Seal
- **Filename**: `signatory-seal.png`
- **Description**: The circular blue stamp with "LOGOGEAR SOLUTION" text
- **Recommended size**: 100x100 pixels
- **Format**: PNG with transparent background

## How to Add Images

1. Save the Logo Gear logo as `logo-gear.png` in this directory
2. Save the signatory seal as `signatory-seal.png` in this directory
3. Restart the backend server
4. Generate DC files - images will be automatically embedded
